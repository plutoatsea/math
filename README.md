### **Math**
---
>This project focuses on building a comprehensive math library from the ground up, incorporating fundamental mathematical concepts, functions, and formulas. The primary goal is to gain a deeper understanding of how mathematical theory translates into functional code.


**Details:**
- _Language_: `C`

- _Build & Run_ :
    ```bash
        gcc main.c -o main && ./main
    ```
- _References_ : [Develop with GCC on Ubuntu](https://documentation.ubuntu.com/ubuntu-for-developers/tutorials/gcc-use/)
